rent
====

endpoints
---------
```
GET  /         --> index.html
POST /mail     --> send mail
GET  /logs     --> logs.html
GET  /logs.csv --> log.csv
```

config
------
```
required vars
-------------

USERNAME - login username
PASSWORD - login password
MAIL - gateway
SENDER - email address
ADDRESS - gateway:port

optional var
------------
PORT - app port
```

--------
`mail.template`
